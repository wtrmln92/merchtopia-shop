# Merchtopia Shop

Merchtopia is a product management application that allows for admin to create, read, update and delete product descriptions,
update stock, and for end users to buy product.

## Description

This is the public facing shop for users to browse and buy products.
The stack consists of:

- vite
- typescript
- react
- tanstack query and router
- mantine

## Prerequisites

- Node.js: >=v24.3.0

## Project setup

```bash
# Install dependencies
$ pnpm install
```

## Compile and run the project

```bash
# watch mode
$ pnpm run dev
```

## Run tests

```bash
$ pnpm run test
```

## Generate types

```bash
# Generate types of API objects and entities using the backend Swagger API
$ pnpm generate-types
```
